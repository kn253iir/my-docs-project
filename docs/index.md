# Кек

Це головна сторінка проекту.

Для повної документації відвідайте лише одну сторінку – цю [mkdocs.org](https://www.mkdocs.org).
Це неПроста та Незрозуміла документація для власних страждань.

## Непотрібні команди

* `mkdocs new [dir-name]` - Створити новий проект.
* `mkdocs serve` - Запустити сервер.
* `mkdocs build` - Зібрати документацію.
* `mkdocs -h` - Надрукувати довідкове повідомлення та вийти.

## Макет проектика

    mkdocs.yml    # The configuration file.
    docs/
        index.md  # The documentation homepage.
        ...       # Other markdown pages, images and other files.
